# Vibespace Changelog

## [2026-03-27] — 카드 폴더블 리디자인 + 상세 스크린 + subDesc + 업체 UI + 캘린더 디폴트

### Added
- 공정 카드 폴더블 UX (READ-ONLY 방식 재설계)
  - 카드 기본 상태: 시공명 + 탐색 chevron(→) + 비용 요약 + 접기/펼기 버튼
  - 시공명 / chevron 클릭 → 상세 슬라이드업 스크린으로 이동
  - 펼쳐진 body: 일정·업체·상세내역 READ-ONLY 미리보기 (수정 불가)
  - `···` 버튼 → 접기/펼기 chevron(∨/∧)으로 교체
- 시공 상세 스크린 (`#screen-proc-detail`): 슬라이드업 애니메이션 (재적용)
  - 탭 구조: **시공 상세 내역** / **시공팁**
  - Back 버튼(←) → 카드 목록으로 복귀
  - `···` 메뉴 → 삭제하기 바텀시트
- DB 구조 확장: 상세내역에 `subDesc`(상세내용) 필드 추가
  - 상세내역 추가/수정 다이얼로그에 상세내용 입력 필드 추가
- 업체 선택 리스트 UI 구조 변경
  - 카드 좌측(선택 영역) / 우측(더보기 버튼) 수직 divider로 분리
  - `event.stopPropagation()`으로 더보기 클릭이 선택으로 전파되지 않도록 처리
- 일정 캘린더 디폴트 월 개선
  - 우선순위: 공사정보 `startDate` → 공정 최소 시작일 → 오늘

## [2026-03-26] — 시공 카드 접기/펼기 + 상세 슬라이드 스크린

### Added
- 공정 카드 접기/펼기 (Collapse/Expand) UX
  - 기본 접힌 상태: 공정명 + 비용 + 날짜 요약만 노출

  - 자세히 보기 → 슬라이드업 상세 페이지로 이동
- 시공 상세 스크린 (`#screen-proc-detail`): 아래에서 위로 슬라이드 애니메이션
  - 탭 구조: **시공 상세 내역** / **시공팁**
  - Back 버튼으로 카드 목록으로 복귀
  - `···` 메뉴: 삭제하기만 노출

## [2026-03-26] — 프로젝트 바텀시트 꽉참 수정

### Fixed
- 모바일 프로젝트 드롭다운 바텀시트가 좌우로 꽉 차지 않던 문제

## [2026-03-25] — 시공일정 선택 UI + 캘린더 개선
