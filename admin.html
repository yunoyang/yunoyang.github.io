GA4 속성 생성 방법
  1. analytics.google.com 접속
  2. 관리 (톱니바퀴) → 속성 만들기
  3. 속성 이름: Vibespace
  4. 업종: 기타 → 비즈니스 목표 선택
  5. 플랫폼: 웹 선택
  6. URL: yunoyang.github.io
  7. 스트림 생성 → 측정 ID (G-xxx) 복사
  8. 위 필드에 붙여넣기 후 저장
⏱ 소요시간: 약 5분 / 완전 무료

Firebase 공동작업 설정
Firebase Console → 프로젝트 설정 → 내 앱 → SDK 구성에서 복사
Firebase 프로젝트 설정 방법
  1. console.firebase.google.com 접속
  2. 프로젝트 추가 → 이름: Vibespace
  3. Google Analytics: 기존 GA4 연결 or 건너뜀
  4. 좌측 메뉴 → Firestore Database
  5. 데이터베이스 만들기 → 프로덕션 모드
  6. 규칙 탭 → 아래 규칙 붙여넣기:
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    match /vb_projects/{id} {
      allow read, write: if request.auth != null;
    }
    match /vb_invites/{id} {
      allow read, write: if true;
    }
  }
}
  1. 프로젝트 설정(톱니) → 내 앱 → 웹 앱 추가
  2. 앱 닉네임: Vibespace → 등록
  3. firebaseConfig 객체 전체 복사
  4. 위 textarea에 붙여넣기 후 저장

구글 시트에서 업체 가져오기
📋 사용 방법
① 구글 시트 1행에 헤더를 아래 형식으로 입력
name, category, area, phone, rating, reviews, description, sourceUrl, contactEmail, reviewLink1, reviewLink2
② 구글 시트 상단 메뉴: 파일 → 다운로드 → Microsoft Excel(.xlsx)
xlsx로 받으면 시트 전체(모든 탭)를 한 번에 가져옵니다
③ 아래에서 파일 업로드 → "전체 교체 저장" → Firestore 자동 업데이트
카테고리: 철거 / 목공 / 타일 / 도배 / 전기 / 설비 / 샷시 / 마루 / 페인트 / 욕실 / 가구 / 청소 / 기타
URL로 직접 가져오기 (공개 시트 전용)
시트를 "링크 있는 누구나 보기"로 설정해도 CORS 제한으로 실패할 수 있습니다.
