사용 방법

    

      
앱에서 아이콘 수정하기

      

        ① 토큰 설정
        Size / Stroke / Color를 조정하면 모든 미리보기에 실시간 반영됩니다
      

      

        ② 아이콘 클릭
        아이콘 카드 클릭 → 하단 코드 패널에서 SVG 코드 확인
      

      

        ③ SVG 복사
        "SVG 복사" 버튼으로 현재 토큰이 적용된 SVG 코드 복사
      

      

        ④ index.html 적용
        복사한 SVG를 index.html의 해당 아이콘 위치에 붙여넣기
